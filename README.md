# WordPress 文章原创角标插件 (Original Badge for WordPress)
![WordPress 版本](https://img.shields.io/badge/WordPress-5.0+-blue.svg) ![PHP 版本](https://img.shields.io/badge/PHP-7.0+-green.svg) ![许可证](https://img.shields.io/badge/license-MIT-yellow.svg)

为 WordPress 文章添加**原创标识角标**的轻量级插件，支持单篇/批量设置原创状态，仅在文章列表、分类页等列表类页面显示角标，单篇详情页不显示，完美适配各类主题的图片容器样式。

## ✨ 核心功能特性
1. **独立插件设计**：不污染主题 `functions.php`，安全易迁移，卸载即恢复原状
2. **精细化权限控制**：每篇文章独立设置原创状态，未勾选绝不显示角标
3. **全场景批量操作**：支持**单篇勾选**、**快速编辑**、**批量设置**（多选文章一键设为原创/取消）
4. **列表页精准显示**：仅在首页、分类页、标签页、存档页、搜索结果页显示角标，单篇详情页隐藏
5. **无冲突样式绑定**：通过专属 CSS 类精准匹配，仅原创文章的 `.ct-image-container` 容器显示角标
6. **后台状态可视化**：文章列表页新增「原创」列，直观显示每篇文章的原创状态

## 📦 安装方法
### 方式1：手动上传（推荐）
1. 前往 [Releases 页面](https://github.com/Mark-ait/Wordpress-Original/releases) 下载最新版本插件压缩包
2. 登录 WordPress 后台 → 外观 → 插件 → 上传插件
3. 选择下载的压缩包，点击「现在安装」，安装完成后**启用插件**
4. 插件自动生效，无需额外配置

### 方式2：本地开发安装
1. 克隆代码到本地：`git clone [https://github.com/your-username/original-badge-plugin.git](https://github.com/Mark-ait/Wordpress-Original.git)`
2. 将整个文件夹上传至 WordPress 站点的 `/wp-content/plugins/` 目录
3. 后台插件页面启用「文章原创角标（带批量设置）」

## 🚀 使用方法
### 1. 单篇文章设置原创
1. 进入后台 **文章 → 所有文章**，点击任意文章进入编辑页
2. 右侧边栏找到「原创设置」面板，勾选「标记为原创文章」
3. 点击「更新/发布」文章，前台列表页自动显示原创角标

### 2. 批量设置原创（核心功能）
1. 进入后台 **文章 → 所有文章**，勾选需要设置的多篇文章
2. 页面顶部「批量操作」下拉框选择「编辑」，点击「应用」
3. 在弹出的批量编辑面板中，找到「原创状态」选项：
   - 选择「设为原创」：批量标记为原创
   - 选择「取消原创」：批量移除原创标识
   - 选择「不修改」：保持原有状态
4. 点击「更新」，批量操作立即生效

### 3. 快速编辑（单篇快捷修改）
1. 进入后台 **文章 → 所有文章**，将鼠标悬停在任意文章标题上
2. 点击弹出的「快速编辑」按钮
3. 在弹出的面板中勾选「设为原创」，点击「更新」即可完成单篇修改

## ❓ 常见问题 (FAQ)
### Q1：角标不显示怎么办？
请按以下步骤排查：
1. 确认文章已**勾选原创**状态，且已**更新/发布**文章
2. 确认当前页面是**列表页**（首页、分类页、标签页等），单篇详情页默认不显示
3. 检查主题是否使用了 `.ct-image-container` 类，若类名不同可自行修改插件 CSS
4. 清除浏览器缓存和 WordPress 缓存（如缓存插件）

### Q2：如何修改角标样式/图片？
直接在插件的 `ob_print_badge_css` 函数中修改 CSS 代码即可：
```css
.ob-original-post .ct-image-container::after {
  /* 可修改宽高、图片地址、位置等 */
  width: 80px;
  height: 80px;
  background-image: url("你的新角标图片地址");
  background-position: top left; /* 改为左上角显示 */
}
```

### Q3：批量设置没生效？
1. 确认已勾选至少一篇文章
2. 确认「批量操作」选择了「编辑」并点击了「应用」
3. 检查是否有足够的编辑权限（管理员角色）
4. 尝试刷新页面后重新操作

### Q4：想在单篇详情页也显示角标？
删除插件中 `ob_add_original_post_class` 函数内的 `if (is_single()) return $classes;` 代码即可。
